Talkprogram
==================
コンピュータと会話形式で晩御飯のメニューを決めるプログラム（？）です。


### 操作の仕方 ###
------
スタートを押すと、名前入力画面になります。名前を入力して、次へ進んでください。  
基本的な流れは、コンピュータが話しかけてきたセリフに対して、画面下のコマンドから自分のセリフをマウスでクリックして選択し、会話を進めていきます。  
選択内容によって、コンピュータからオススメされる料理が変わります。
